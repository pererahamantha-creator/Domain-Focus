# Domain Focus

A gamified productivity and study planner inspired by anime-style mission systems.

Domain Focus helps students stay organized by turning daily study tasks into missions, rewarding progress with Energy, Rank progression, streaks, and focus sessions.

## Features

- ✅ Daily Mission Tracker
- ⚡ Energy Point System
- 🏆 Rank Progression
- 🔥 Daily Streak Tracking
- ⏱ Domain Focus Timer (Pomodoro)
- 📊 Study Statistics
- 💾 Local Storage Persistence
- 📱 Responsive Mobile & Desktop Design
- 🌙 Modern Dark Theme UI

## How It Works

1. Add study missions.
2. Complete missions to earn Energy.
3. Increase your Rank as your Energy grows.
4. Maintain daily streaks.
5. Enter Domain Focus mode for distraction-free study sessions.

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser Local Storage

No backend, database, authentication, or internet connection required.

## Installation

### Option 1: Run Locally

Download the repository and open:

